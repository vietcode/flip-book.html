The Making Of

We start by writing the basic markup for the book. We use <section> to represent a single book page within the component.

In browsers with Javascript disabled, the content is simply displayed linear. Javascript will transform the content into a flippable book.

We then use some styles to make the content looks nicer, using boom.css (http://alistapart.com/article/boom)
